### About

Utility to convert .vtt files to .srt format.

### Installation

```npn -g i srt-from-vtt```

or

```yarn global add srt-from-vtt```

or build an executable file using [pkg](https://www.npmjs.com/package/pkg)

```yarn makebin```

### Usage

```srt-from-vtt <folder(s)>```
